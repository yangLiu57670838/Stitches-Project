Thank you for using Stitches

What is Stitches?

Stitches is an easy to use website for image uploading, DTS downloading and service fee paying.

How to upload DTS images?

Admin can click each order listed in your dashboard, then choice Upload DTS Files function.

How to receives notification?

Admin will receive an email once users make an new order, admin will receive both emails and inbox messages after users pay the service fee.

How to manage users and orders?

Admin can click the MANAGE USERS button on the left menu bar to view current users, add new users, change users info, and delete users. Admin can also click each order listed in your dashboard to view orders details, download uploaded DTS files, and delete orders.
